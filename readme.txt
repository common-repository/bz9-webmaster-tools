The BZ9 WordPress Plugin simplifies the installation and management of over 17 affiliate and internet marketing tools for your blogs and WebPages.

== Description ==

The plugin will install within seconds, it's dead simple ... and once installed you will see an orange circle at the top of the HTML editors for both pages and blog posts.

Click the Orange circle so that you may add the BZ9 Webmaster Tools code.

Management pages area also added to the left hand navigation within your WordPress installation.

By adding BZ9 Webmaster Tools code you will be able to add any of the following tools and services which are instantly switched on and off at the click of your mouse:

* Designer Dropins
* Custom Dropins
* GEO Targeting
* GEO Tagging
* Peel Aways
* Navigation Bars (free)
* Viral bars (free)
* Pop Unders
* Exit Pop Ups
* Delayed Dropins
* Optin forms
* Follow Me Tabs (free)
* Optin Bars (free)
* Stealth Tracking
* Pop Unders
* Dynamic Ads
* Over 25 ‘Virtual Presenters’

Custom dropins is a very comprehensive tool which allows you to place any content any where on the page from banners to optin forms and you can also delay their appearance to a set time interval or to appear on exit. Great for offering last chance offers and CPA deals ...

The BZ9 WordPress Plugin contains the following key features:

* Simple form for adding new BZ9 tools and presenters
* Easy management for editing and deleting existing tools
* Add notes and descriptions for easy recognition
* Easy navigation back to BZ9 for management of the tools created
* Over 20 video tutorials installed and viewable within the plugin
* First class support
* Access to other free BZ9 Internet & Affiliate marketing services

The tools listed as free are free to use, the others will require a small upgrade fee.

For more information please go to: http://bz9.com/plugin


== Installation ==
1. Upload the files to the /wp-content/plugins/bz9_webmaster_tools/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
